# SMB AI Maturity Assessor

`smb-ai-maturity-assessor` applies the companion SMB AI Maturity Model to a company, team, function, or workflow.

Use it when someone asks where they are in AI maturity, what is blocking the next level, or what to do next.

## Install

Copy this directory into your agent skill directory:

```text
skills/smb-ai-maturity-assessor/
```

For best results, keep the companion framework files in `frameworks/smb-ai-maturity-model/`.

## Try It

```text
Use smb-ai-maturity-assessor to assess our support workflow. We use AI for drafting responses, but review is inconsistent and we do not measure quality yet.
```

## Good Outputs

A good answer:

- narrows the assessment scope
- separates dimension scores from spine scores
- applies the governance and accountability cap
- names the binding constraint
- recommends the smallest next maturity move
- states confidence and evidence gaps
